Logic Expression Evaluator

How to Use the Logic Expression Evaluator

This tool allows you to evaluate and compare logical expressions. Here's how to use it:

1. Evaluate: Enter a logical expression (e.g., "a&b|c") and click "Evaluate" to see its truth table.
2. Compare: Click "Compare", enter two expressions, and click "Compare" again to check if they're equivalent.
3. Apply Where: Click "Apply Where", enter an expression and conditions, then click "Apply Where" again to filter the truth table.

Operators: & (AND), | (OR), ^ (XOR), ~ (NOT), - (IMPLIES), = (EQUALS)

Variables can be any lowercase letter. The tool will generate all possible combinations of TRUE (1) and FALSE (0) for each variable in your expression.
